# Kava Wiki 2.0

A comprehensive multilingual knowledge portal about Piper methysticum (Kava).

## Features
- 22 language versions (DE, EN, ES, FR, NL, PL, CS, PT, IT, RO, HU, BG, EL, TR, NO, DA, FI, SV, JA, ZH, RU, KA)
- Scientific information about effects, safety, and ingredients
- Cultural knowledge about ceremonies and traditions
- Interactive glossary with 50+ terms
- Kava bar finder with world map
- Dosage calculator
- Study database with 100+ sources

## Tech Stack
- React 19 + Vite + TypeScript
- Express + tRPC backend
- TailwindCSS + Radix UI
- Drizzle ORM
- OpenAI API for translations

## Development
```bash
pnpm install
pnpm dev
```
